Understanding Flex

flex concept: flex works on two axis concepts. main axis and the cross axis. this axis is defined by the flex direction of either row(East/West) or (North/South) column. the cross is assumed based on the flex direction

flex -container- properties
display: flex;
*flex-direction: row/column changes how the flex x and y axis function (sets the main axis and the cross axis is derived from it) row is default column; so the items stack vertically - use row to stack horizontally - very powerful attribute!
*flex-wrap: (nowrap|wrap|wrap-reverse|initial|inherit) whether items in a row can start a new row
flex-flow: ( *flex-direction* *flex-wrap* |initial|inherit;) shorthand for flex-direction and flex-wrap on one line - flex-wrap and flex-direction are therefore not required!
justify-content: (start|center|space-between|space-around|space-evenly) how the space in the main-axis is used up (either N/S or E/W depending on flex direction of row or column
align-items: (stretch|flex-start|flex-end|center|baseline) how the space in the cross-axis is used up
align-content: (stretch|flex-start|flex-end|center|space-between|space-around) how to align the ROW (or column) of items (assuming multiple items in a row or column)
-everything in a flex container is a flex item

flex -items- properties
order: (integer|initial|inherit) 0 is default // ignores html layout order and repositions a flex item's order within the flex container
flex-grow: (integer|initial|inherit) how to fill remaining space (default is 0 which means it does NOT take up remaining space in row or column) it is units/ specifying 2 is not twice as big// it uses 2 more equal units of space
flex-shrink: (integer|initial|inherit) default is 1 (all shrink at equal rate) higher number mean item will shrink at a faster rate (and be smaller within its axis)
flex-basis: (integer|initial|inherit) size of items (if width: is not specified - can be used instead of width) limited by max-width and min-width use with flex-shrink:1; (or flex-grow) to prevent overflow of items width
flex: (flex-grow flex-shrink flex-basis|auto|initial|inherit;) shorthand for flex-grow, flex-shrink and flex-basis; default is flex:0 1 auto; flex: 1 0 200px;
align-self: (auto|stretch|center|flex-start|flex-end|baseline|initial|inherit) to position a single item in a row or column
